Polynomials

In a polynomuial "x" the highest power of that polynomial is called degree of polynomial
for example in the polynomial 4x + 2 the degree is '1'.
Similarly, in polynomial 5x2 + 2x + 3 , the degree of polynomial is 2

 In this way we can say,
  • A polynomial with degree "1" is a Linear polynomial
  • A polynomial with degree "2" is called Qudratic poynomial
  • A polynomial with degree "3" is called Cubic polynomial
If p(x) is a polynomial in x, and if k is any real number, then the value obtained by replacing x by k in p(x), is called the value of p(x) = k and is denoted by p(k)

    Algebric methods of solving a pair of linear equations in two variables

    Substitution method:

    We shall explain the substituion method with the help of some examples:

    Solve the following pair of linear equations by substitution method:

    7x - 15y = 2 (1)
    x + 2y = 3 (2)

     Solution

    Step.1 : We pick either of the equations and write one variable in terms of the other.
    Let us consider equation (2):

    x + 2y = 3 and write it as,
    x = 3 - 2y (3)

    Step.2 : Substitute the value of x in equation (1), we get

    7(3 - 2y) - 15y = 2
    21 - 14y - 15y = 2
    -29y = -19

    Therefore, y = 19/29

    Step.3 : Substituting this value of y in equation (3), we get
    x = 3 - 2(19/29) = 49/29

    Therefore the solution is x = 49/29 and y = 19/29

    Verification: Substituting x = 49/29 and y = 19/29, you can verify that both the equations (1) and (2) are satisfied

    Remark: We have substituted the value one variable by expresing it in terms of the other variable to solve the pair of linear equations. That is why the method is known as substitutution method

    Elimination method:

    Now let us consider another methiod of eliminating (i.e remeoving) one variable. This is sometimes more convenient than the substitution method. Let us see how it works

    Solving the equations by elimination method:

    9x - 4y = 2000 (1)
    7x - 3y = 2000 (2)

    Solution:

    Step.1 :     Multiply equation (1) by 3 and equation (2) by 4 to make the coefficients of y equal. Then we get equations:

    27x - 12y = 6000 (3)
    28x - 12y = 8000 (4)

    Step.2: Subtract equation (3) from equation (4) to eliminate y , because the coefficients of y are the same. So we get,

    (28x - 27x) - (12y - 12y) = 8000 - 6000
    x = 2000

    Step.3: Substituting the value of x in equation (1), we get
    9(2000) - 4y = 2000
    y = 4000

    Therefore, the solutions of x and y is x = 2000, y = 4000

    Quadratic Equations


    We know that the standard formula for a polynomial p(x) = ax2 + bx + c = 0, a ≠ 0

    We have learnt in previous years how to find the roots of a poylnomial by splitting the middle term, now let us learn a new way to solve these sums:

    Consider the equation ax2 + bx + c = 0 (a ≠ 0). Dividing throughout by a, we get:

    x2 + b/a x + c/a = 0

    This is the same as (x + b/2a)2 - (b/2a)2 + (c/a) = 0

    i.e, (x + b/2a)2 - b2 - 4ac/4a2 = 0

    So the roots of the given equation are the same as those of,

    (x + b/2a)2 - b2 - 4ac/4a2 = 0, i.e those of (x + b/2a)2 = b2 - 4ac/4a2

    If b2 - 4ac ≥ 0, then by taking the square roots in (1), we getx + b/2a = ± √b2 - 4ac/2a

    Therefore, x = -b ± √b2 - 4ac/2a

    So the roots of ax2 + bx + c = 0 are -b + √b2 - 4ac/2a and -b - √b2 - 4ac/2a ,
    If b2 - 4ac ≥ 0 = If b2 - 4ac < 0, the equation will have no roots

    This, if b2 - 4ac ≥ 0, then the roots of the quadratic equation
    ax2 + bx + c = 0 are given by -b ± √b2 - 4ac/2a

    This formula for finding the square roots of thr quadratoc equation is known as the quadratic formula

    Nature of the roots:

    i) Two distinct roots real roots, if b2 - 4ac > 0
    ii) two equal roots, if b2 - 4ac = 0
    iii)No real roots, if b2 - 4ac < 0
